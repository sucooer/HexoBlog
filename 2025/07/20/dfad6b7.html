免费版claude-code！｜rovo-dev-cli-安装到实战完整教学！

若雨2025-07-20


前言

这款由开发工具巨头Atlassian（也就是Jira 和Trello 的母公司）推出的命令列Agent 工具，不仅功能强大，而且目前处于Beta 阶段，可以让开发者完全免费使用！

最吸引人的是，它主要使用的模型是Anthropic 的Claude Sonnet 4，并且每天提供高达2000 万个tokens的免费额度，做一些中小型的专案基本上是用不完的。今天我们就来从零开始，一步步完成安装设定，并透过实战修改一个旧专案，让你快速上手这款强大的工具。

Rovo Dev CLI 亮点分析

  • 佛心公司的慷慨之作：由Atlassian 出品，品质有保证。
  • 强大的AI 模型：主要使用Claude Sonnet 4，并会视情况切换至Sonnet 3.7 或3.5v2，程式撰写能力经过实测优于Gemini 1.5 Pro。
  • 海量免费额度：目前处于Beta 阶段，每日提供20M（2000 万）tokens，对于绝大多数开发任务都绰绰有余。
  • 命令列Agent：能直接在你的终端机中与AI 互动，执行指令、修改档案，提升开发效率。

安装教学

可参考官方文档！详细安装步骤往下看！

注册账号

首先，我们需要前往Rovo Dev CLI 官方页面并注册一个Atlassian 帐号。好消息是，它支援直接使用Google 帐号登入，非常方便。

注册Atlassian

或者直接点击[此链接](https://www.atlassian.com/solutions/devops/ai-innovation)，登录后会跳转到创建网站页面，输入一个未使用的名称，点击Agree and start now，如果无法创建网站，请尝试更换代理。

创建网站

下载并安装CLI程序

注册完毕后，我们需要下载程序。官方提供了二进制文件，Windows 系统可以直接下载EXE文件。

可以使用终端下载或者直接复制EXE地址进行下载

下载exe文件

下载完成后，将应用程序放到一个你喜欢的位置。为了方便，你可以：

  • **方案A (最简单)**：将.exe文件直接丢到C:\Windows\System32文件夹，此路径已在系统环境变量中，可直接在终端使用。
  • **方案B (个人偏好)**：在C:\Program Files下建立一个新文件夹(例如rovodev-cli)，将.exe放入，然后手动将此路径加入到系统的Path环境变量中。

获取Token并登录

接下来，我们需要创建一个API Token 来让CLI 工具可以验证你的帐号。

  1. 登录你的Atlassian 帐户后台，找到API Token 的页面。
  2. Atlassian 基于安全考量，不允许产生永久的Token，最长有效期为一年。
  3. 为你的Token 取一个方便识别的名称，然后点击创建。
  4. 重要：创建后请立即复制Token，这个窗口关闭后就无法再次查看。

总结

前后试用过了RovoDevCLI和Gemini CLI ，个人感觉Rovo Dev CLI 在撰写的体验上是优于Gemini CLI 的。很多的修改指令我几乎都只要讲一次，它就能准确完成，而且产出的错误率非常低。

这可能不完全是模型能力的差异，更可能是Rovo Dev CLI 在提示词工程(Prompt Engineering)以及工具调用格式上做了更好的优化，让AI 的表现更稳定、更符合预期。

总之，这是一款目前非常值得开发者们尝试的良心工具，现阶段免费 ，当然不会是永久免费，或许试用额度也会有所限制。 可以的话，大家趁现在玩玩看吧！ 如果能让自己的工作效率有所提升就再好不过。对于自己的需求也会更加了解，方便在选择 AI 产品时能找到最适合自己的。
